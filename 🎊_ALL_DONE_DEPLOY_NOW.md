### Quick Start:
```bash

# 2. Deploy backend
cd backend
vercel

# 3. Add environment variables

# 5. Deploy frontend

vercel

# 6. Add frontend env
vercel env add REACT_APP_API_URL

# 7. Deploy frontend
vercel --prod

# 8. Update backend FRONTEND_URL

vercel env rm FRONTEND_URL
vercel env add FRONTEND_URL
vercel --prod
```
